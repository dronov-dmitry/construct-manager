# ConstructManager

https://dronov-dmitry.github.io/construct-manager/

**ConstructManager** — кроссплатформенное мобильное и десктопное приложение для управления строительными проектами. Построено на Flutter с бэкендом на Supabase.

### Возможности

- **Управление объектами** — создание, редактирование, отслеживание этапов строительства
- **Бюджет** — учёт затрат по каждому объекту
- **Фотофиксация** — привязка фотографий к объектам и этапам
- **График работ** — планирование и контроль сроков
- **Ответственные лица** — назначение и контроль исполнителей
- **Информация** — заметки и дополнительные данные по объекту
- **Карта** — отображение объектов на карте
- **Биометрия** — вход по отпечатку пальца / Face ID
- **Локализация** — русский и английский языки
- **Тёмная тема** — поддержка светлой, тёмной и системной темы

### Стек

- **Frontend:** Flutter (Dart)
- **Backend:** Supabase (PostgreSQL, Auth, Storage, RLS)
- **База данных:** PostgreSQL через Supabase
- **Сборка:** Flutter build для Android, iOS, Windows, macOS, Linux, Web
